# Games Showcase

This folder contains several browser games and a unified launcher page:

- Open `index.html` to access all games from one place.
- Each game also remains available in its own folder.

## Recommended Run Method (Windows)

1. Double-click `start-games.bat`
2. Your browser should open automatically at:
   - [http://localhost:8000](http://localhost:8000)

## Manual Run Method

From this folder (`Games`), run one of these:

```powershell
py -m http.server 8000
```

or:

```powershell
python -m http.server 8000
```

Then open:

- [http://localhost:8000](http://localhost:8000)
